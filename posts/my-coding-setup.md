## Hardware

I primarily do most of my coding on my Macbook Air M2. It's a fantastic laptop
for coding (as well as schoolwork), and I can do everything I need to with ease.
Sometimes I use my old Microsoft Surface Laptop Go with Arch Linux and Hyprland,
but it's not common for me to use it. As per my headphones, I use the AirPods
Pro 2. It has great audio quality, bass, and noise cancellation — A must have
for coding especially in distracting environments.

I use Zed editor. It is fast, good looking and feature-full. If you want to know why I switched from Neovim, read [this article](https://smit.codes/writing/switching-to-zed).

For fonts, I prefer the JetBrainsMono Nerd Font. In my experience, it's an
amazing font that's highly readable and aesthetically pleasing. I use the Nerd
Font simply because I use Neovim and require Nerd Fonts for my setup.

I usually use either the VSCode Dark Modern theme or my own Shale theme.

### Terminal

I use the [Ghostty](https://ghostty.org) terminal with
[starship](https://starship.rs) prompts. I usually use the
[fish shell](https://fishshell.com)

### Browser

~~I use [Zen](zen-browser.app) browser. There's no real reason why, I just
wanted a Firefox based browser and Zen looked pretty nice.~~

I use [Helium Browser](https://helium.computer). It's based on chromium, it's
fast and its bloat-free.

### Design

Whenever I need to work on designing UI, I use [Figma](https://figma.com).

## Languages

For most of my work, I use Rust as my preferred programming language (CLIs,
utilities, backend etc.). When I'm working on websites, I tend to use SvelteKit.
I've tried React and hated it, and so Svelte seems to be the perfect web
framework for me. In fact, this website is made with a Svelte front-end and a
Rust powered backend!

I also use Java when Minecraft modding.
